# Kiloword

## Usage
Processing and analysing kiloword dataset 

## Installation and dependencies
To clone the repo:
```
    git clone https://github.com/victoria-brami/kiloword.git
```
Then installing the packages:
```
    pip install -r requirements.txt
```
